= Quick Start

Apply the `fr.rozanc.helm` plugin to your Gradle project:

[source,groovy,role="primary",subs="+attributes"]
.Groovy
----
plugins {
    id 'fr.rozanc.helm' version '{project-version}'
}
----

[source,kotlin,role="secondary",subs="+attributes"]
.Kotlin
----
plugins {
    id("fr.rozanc.helm") version "{project-version}"
}
----
